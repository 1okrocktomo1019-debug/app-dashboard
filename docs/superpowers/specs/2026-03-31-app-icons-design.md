# アプリアイコン デザインスペック

**日付**: 2026-03-31  
**対象**: 1日1アプリ ダッシュボード（app-dashboard）

---

## 概要

各アプリカードの左上にSVGアイコンを追加する。カードデザイン（白カード・#1a1a2e ヘッダー・アクセント #4f6ef7）に映えるグラデーション背景 + 白アイコンスタイル。

---

## デザイン仕様

### 共通ルール
- **形式**: SVG
- **サイズ**: 52×52px（viewBox 0 0 52 52）
- **角丸**: rx="12"
- **スタイル**: グラデーション背景 + 白（rgba）アイコン
- **配置先**: `static/icons/`

---

### 各アイコン詳細

#### 1. チャイドル (`chaidol.svg`)
- **グラデーション**: ピンク #f472b6 → パープル #a855f7（左上→右下）
- **モチーフ**: 子供シルエット（頭＋体）の上にTシャツが重なる「バーチャル試着」イメージ
- **構成要素**:
  - 白い丸（頭）: cx=26, cy=14, r=6
  - 薄い楕円（体のベース）: 不透明度25%
  - Tシャツ形のパス（衿・袖・胴体）: 不透明度85%
  - スパークル（右上）: バーチャル感を表現

#### 2. X→Threads リライトツール (`x-to-threads.svg`)
- **グラデーション**: ブルー #4f6ef7 → シアン #06b6d4（左上→右下）
- **モチーフ**: 斜め配置のペン（リライト・書き直しのイメージ）
- **構成要素**:
  - ペン本体ポリゴン: (35,9)→(43,17)→(23,37)→(15,29)
  - キャップ帯: 上端に別ポリゴン（不透明度55%）
  - ペン先三角: (15,29)→(23,37)→(9,45)
  - インクドット: ペン先に小円
  - 水平ライン2本（右下）: リライトを示す罫線

#### 3. 問い合わせフォーム自動送信ツール (`form-sender.svg`)
- **グラデーション**: オレンジ #f97316 → レッド #ef4444（左上→右下）
- **モチーフ**: 封筒 + 送信完了チェックバッジ
- **構成要素**:
  - 封筒本体（白・角丸rect）
  - 封筒フラップ（V字パス）
  - 赤いサークルバッジ（右下）: 送信完了を表現
  - バッジ内チェックマーク

## HTMLテンプレートの変更

`templates/index.html` のカードに `<img>` タグでアイコンを追加する。

```html
<a class="card" href="{{ app.url }}" target="_blank" rel="noopener noreferrer">
  {% if app.icon %}
  <img src="{{ url_for('static', filename='icons/' + app.icon) }}" width="52" height="52" alt="{{ app.name }} icon">
  {% endif %}
  <h2>{{ app.name }}</h2>
  ...
</a>
```

`apps.yaml` に `icon:` フィールドを追加：

```yaml
- name: チャイドル
  icon: chaidol.svg
  ...
- name: X → Threads リライトツール（仮）
  icon: x-to-threads.svg
  ...
- name: 問い合わせフォーム自動送信ツール（仮）
  icon: form-sender.svg
  ...
```
